# Peak Innovation Studios Homebrew Tap

Casks for apps published by [Peak Innovation Studios](https://github.com/Peak-Innovation-Studios).

## Usage

```sh
brew tap peak-innovation-studios/tap
brew install --cask gpg-manager
```

## Available Casks

| Cask          | Description                                                   |
|---------------|---------------------------------------------------------------|
| `gpg-manager` | Modern macOS GUI for managing GPG keys, gpg-agent, and Git signing |
